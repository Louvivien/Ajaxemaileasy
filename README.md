###### Étapes à suivre en local
* Faite un git clone: https://github.com/MelloFlo/Ajax-c-cool
* `bundle install`
* `rails db:create`
* `rails db:migrate`
* `rails db:seed`
* `rails s`
* Et faites vos tests !
